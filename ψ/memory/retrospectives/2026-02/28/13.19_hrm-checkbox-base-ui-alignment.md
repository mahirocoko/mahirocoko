## Lessons Learned

- **Base UI uses `data-checked`, Radix uses `data-state="checked"`** — They're different libraries with different attribute conventions. shadcn's custom variant supports both, but syntax differs.
- **Check shadcn/tailwind.css before assuming variant syntax** — The custom variants are documented there.
- **Edit tool can corrupt files with multiple passes** — Prefer atomic replacements or `write` when file is broken.
