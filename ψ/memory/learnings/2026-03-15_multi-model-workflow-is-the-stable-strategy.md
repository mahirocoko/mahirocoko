# Multi-model workflow is the stable strategy

## Pattern

No single AI model wins across all dimensions. The optimal workflow uses different models for different tasks, and this isn't a temporary gap — it's structurally stable because each lab optimizes for different strengths.

## Evidence (from two Theo videos, 2026-03-15)

| Task | Best Model | Why |
|------|-----------|-----|
| Backend/full-stack coding | GPT 5.4 | Best reasoning, token efficiency, instruction following |
| Frontend UI design | Opus 4.5 + skill | Iterates on feedback, avoids generic patterns |
| Initial page scaffold | Gemini 3 Pro | Good defaults, wide design range |
| UI refinement | Opus 4.5 | Understands when to drop libraries, proposes structural changes |
| Hard reasoning problems | GPT 5.4 Pro | Solves previously unsolvable puzzles |

## Amplifier

Skills-as-context (markdown files) work universally across all models. They're the multiplier that lifts any model's output quality. The skill matters more than the model choice.
